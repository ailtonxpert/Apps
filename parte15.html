Cypress

Tutorial

1. No prompt de comando, crie uma pasta com o seguinte comando: mkdir nomeDaSuaPasta
2. Abra sua pasta com o seguinte comando: cd nomeDaSuaPasta.
3. Digite o seguinte comando para instalar a pasta npm_modules na sua pasta: npm install cypress --save-dev.
4. Digite o seguinte comando para abrir o Cypress: npx cypress open
5. Abra a pasta cypress e procure a pasta 'integration'.
6. Dentro da pasta 'integration', crie um arquivo com a seguinte extensão: nomeDoSeuArquivo.spec.js
7. Procure o seu arquivo na página do Cypress que foi aberta no passo 4 e clique sobre ele.
8. Digite o código de teste no seu arquivo e salve.
